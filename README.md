# Harness and Loop

This repo is based on Sean's YouTube explainer on agent harnesses, loop engineering, evals, and LLM Ops:

https://www.youtube.com/watch?v=GrNbuWWJYiI&t=43s

The project contains:

- `harness_llm_ops_notes.html`: an interactive reconstruction of the video/diagram.
- `harness_llm_ops.png`: the source diagram used by the notes.
- `agent_harness/`: a small local Python harness implementing the blueprint with memory, tools, guardrails, traces, evals, and a release gate.

Raw transcript and summary `.txt` files are intentionally not tracked.
